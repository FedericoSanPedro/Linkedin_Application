# Welcome to the Linkedin Application

You will be able to receive the data from an user's profile only by inseting the url of his linkedin profile.

1. Git clone the project

2. Create a file called .env like .env.example and add the required needs for the project. In this case API_URL is your local IP.
And other one in scraper that contains your linkedin user and password.

3. Update the modules

   ```bash
   npx expo install
   npm install
   ```

   /scraper

   ```bash
   npm install
   ```

4. Start the app

   ```bash
   npx expo start
   ```
   /scraper

   ```bash
   node server.js
   ```

5. Open the website from npx expo start and insert the url.
